Valor_prod : Valor del producto

Cve_mon : Clave de la moneda de facturación (1=pesos)
Tip_cam : Tipo de cambio aplicado en la factura
Unidad: Unidad base del producto
No_ped : Número de pedido del cliente
No_rem : Número de remisión del cliente
Cve_suc : Clave de sucursal
Lote : Lote de producto

Nom_fac : Nombre de facturación del cliente
